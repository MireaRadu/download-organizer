Configuration

Here you can add rules for redirecting downloads based on file type. Some file types are more generic than others (e.g. images is more generic than Png, so priority matters and can be adjusted in the Actions column.

Due to security reasons, you can only specify paths relative to your download directory. To go around that, you can edit your download location.

Advanced

If you can't find the desired file type, you can add your own by specifying a MIME type (e.g. image/jepg) and a label. The newly added file type will be visible in Configuration, under Custom.
